Odoo的docker安装
----------------

1. 从docker.com下载安装docker
2. 从https://git-scm.com/downloads 下载安装版本控制工具git
3. `git clone https://github.com/odoo-china/odoo-training`
4. `cd odoo-training`
5. `docker-compose up -d` 大约15分钟
6. 在浏览器中访问 http://localhost:8069
7. 搞定
